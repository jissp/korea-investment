당신은 NestJS로 구성된 현재 프로젝트에 RSS 피드를 수집하는 기능을 구현해야 합니다.

# 기능 요구사항
1. RSS(XML)를 조회합니다.
2. 조회한 RSS 피드를 파싱하여 객체로 변환하고 응답합니다.

# 아키텍처

## XmlParser

1. fast-xml-parser를 Wrapping 한 모듈
2. Service Layer에서 xml을 파싱해서 객체로 변화하는 로직만 제공.

## RssReader

1. 사용자가 전달한 RSS URL을 조회
2. XmlParser를 통해 파싱한 데이터를 응답
3. 응답데이터는 Generic으로 전달받아 Type 체크

# 작업 방식
1. 위와 같이 작업을 진행하려고 하는데, 내가 놓치거나 추가적인 아이디어가 있다면 제시해줘.
2. 너의 아이디어나 추가적인 제안이나 질문을 한다면 진행하라고 명확하게 이야기할 때 까지 나와 소통해줘.
3. 작업을 진행할 때 반드시 플랜 모드는 사용하지 말고, 위에서 이야기한 내용을 기반으로 작업을 진행해줘.
